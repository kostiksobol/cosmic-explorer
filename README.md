## 🚀 Deployment Instructions

### Prerequisites
- GitHub account
- Git installed on your computer
- Repository set up

### Steps to Deploy

1. **Commit your files**:
   ```bash
   git add .
   git commit -m "Add cosmic explorer website"
   git push origin main
   ```

2. **Enable GitHub Pages**:
   - Go to your repository on GitHub
   - Click **Settings** tab
   - Scroll to **Pages** section
   - Select **Deploy from a branch**
   - Choose **main** branch
   - Click **Save**

3. **Access your site**:
   - Your site will be available at: `https://your-username.github.io/your-repository-name/`
   - It may take a few minutes to deploy

### Custom Domain Setup (Optional)

1. **Configure DNS**:
   - Add a CNAME record pointing to `your-username.github.io`
   - Or add A records pointing to GitHub Pages IPs

2. **Update CNAME file**:
   - Edit the `CNAME` file in your repository
   - Add your custom domain (e.g., `cosmic-explorer.com`)

3. **Enable HTTPS**:
   - In GitHub Pages settings, check **Enforce HTTPS**

## 🔧 Local Development

### Running Locally

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/demo-repository.git
   cd demo-repository
   ```

2. **Open in browser**:
   - Simply open `index.html` in your web browser
   - Or use a local server:
     ```bash
     python -m http.server 8000
     # Then visit http://localhost:8000
     ```

### Making Changes

1. **Edit files**: Modify HTML, CSS, or JavaScript files
2. **Test locally**: Check changes in your browser
3. **Commit changes**: Push to GitHub to update live site
